## 📌 Overview

The **Heart Disease Prediction System** is a machine learning-powered web application that predicts 
whether a patient is likely to have heart disease based on clinical medical parameters. 
Built with **Logistic Regression** and deployed via a clean **Streamlit** interface, 
this project demonstrates an end-to-end ML pipeline — from raw data to an interactive prediction tool.

> 🏥 This system can assist medical professionals and individuals in early detection 
> of heart disease risk, enabling timely medical intervention.

---

## 🎯 Problem Statement

Heart disease is one of the leading causes of death globally. Early detection through 
data-driven methods can significantly improve survival rates. This project aims to:

- Analyze clinical attributes of patients
- Train a machine learning model to classify heart disease presence
- Provide an easy-to-use web interface for real-time predictions
